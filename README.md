# 小学生错题高效学习智能体

错题高效学习智能体，帮助小学生有效提升知识掌握。

## 功能特性

- ✅ 错题录入（拍照识别）
- ✅ AI智能讲解
- ✅ 智能练习
- ✅ 学习进度追踪
